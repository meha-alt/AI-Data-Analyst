# AI-Data-Analyst
This project is an AI-powered data analysis system designed to transform raw Excel datasets into actionable business insights. It automates Exploratory Data Analysis (EDA), detects anomalies, evaluates data quality, and integrates with a Large Language Model (LLM) to generate decision oriented recommendations.

The system bridges the gap between raw data and meaningful business understanding, making it useful for analysts, students, and decision-makers.

Tech Stack
Python,Pandas, NumPy,Ydata-profiling,LLM Integration (for insights),Streamlit (UI),Render (Deployment)
